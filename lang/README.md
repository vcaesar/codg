## Contributing Translations

The English [`README.md`](../README.md) at the repository root is the source of truth. When updating content, please update the English version first, then propagate changes to the translated files in this directory.

File naming convention: `README.<lang>.md` where `<lang>` is an ISO 639-1 code (e.g., `zh`, `ja`, `fr`). Traditional Chinese uses the custom suffix `zht`.
